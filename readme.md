# Projeto de estudo em construção ⚠️

## Instruções:

### Rodar o projeto após clone:

#### Backend:

- `cd backend`

- `pnpm run dev`

#### Frontend:

- `cd frontend`

- `pnpm i`

- `pnpm start`

---

### Tecnologias utilizadas:

- ReactJs

- Graphql

- Typescript

- Material Ui

- MongoDB Compass

- Styled Components
